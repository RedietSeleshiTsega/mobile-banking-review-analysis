# Mobile Banking App Reviews Analysis

## Project Overview
This project scrapes and analyzes user reviews from the Google Play Store for three Ethiopian banks:
- Commercial Bank of Ethiopia (CBE)
- Dashen Bank
- Bank of Abyssinia (BOA)

The goal is to analyze customer sentiment, identify satisfaction drivers and pain points, and provide actionable insights for app improvement.
